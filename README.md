# piyolog-viz

ぴよログのエクスポートデータをPythonで解析し、ブラウザで見られるHTMLダッシュボードを生成するスクリプトです。

## できること

- **成長グラフ** — 身長・体重の推移
- **睡眠グラフ** — 睡眠時間の推移
- **授乳グラフ** — 母乳時間・ミルク量の推移
- **排泄グラフ** — おしっこ・うんち回数の推移
- **うんちアラート** — 最後のうんちから何日経ったかを色分け表示
- **タイムライン** — 1日の睡眠・授乳・排泄を帯グラフで可視化（保育園の送迎時間にも対応）
- **離乳食マップ** — 月齢ステージ別の食材進捗・アレルゲン色分け・次におすすめの食材提案

## 必要なもの

- Python 3.8 以上（標準ライブラリのみ使用、追加インストール不要）
- ぴよログのテキストエクスポートファイル

## セットアップ

### 1. 設定ファイルを作成する

```bash
cp config.json.example config.json
```

`config.json` を開いて、お子さんの情報を入力します。

```json
{
  "birth_date": "2025-10-01",
  "baby_name": "たろう",
  "output_filename": "piyolog_viz.html"
}
```

| 項目 | 説明 |
|------|------|
| `birth_date` | 生年月日（YYYY-MM-DD形式） |
| `baby_name` | ダッシュボードのタイトルに表示される名前 |
| `output_filename` | 生成されるHTMLのファイル名（変更不要） |

### 2. ぴよログのデータを配置する

ぴよログアプリの「設定」→「記録のエクスポート」からテキストファイルをエクスポートし、`data/` フォルダに置きます。ファイル名は `ぴよログ_YYYY-MM.txt` の形式にしてください。

```
data/
├── ぴよログ_2025-10.txt
├── ぴよログ_2025-11.txt
└── ぴよログ_2026-01.txt
```

動作確認には `data/sample.txt`（架空データ）を使えます。

### 3. 実行する

```bash
python3 piyolog_viz.py
```

`piyolog_viz.html` が生成されます。ブラウザで開いてください。

## サーバーモードで使う（ファイルを追加したら自動更新）

```bash
python3 server.py
```

`http://localhost:8765` をブラウザで開きます。`data/` フォルダに新しいエクスポートファイルを置くと、約3秒後に自動で再生成・ブラウザリロードされます。

## 離乳食マップを使う

離乳食の記録はぴよログに以下の形式でメモしておくと、スクリプトが自動で読み取って「クリア済み」に反映します。

```
HH:MM 離乳食 にんじん
HH:MM 離乳食 おかゆ・じゃがいも
```

または `ryunyu_foods_example.json` をコピーして手動で記録することもできます。

```bash
cp ryunyu_foods_example.json ryunyu_foods.json
```

食材の導入日・反応（「すき」「ふつう」など）を記入すると、マップのホバー表示に出ます。

## 保育園の送迎時間をタイムラインに表示する

ぴよログの「カスタム項目」から「いってきます」「ただいま」という項目を作って記録すると、タイムラインに保育園帯（黄色）が表示されます。

## ファイル構成

```
piyolog-viz/
├── piyolog_viz.py          # メインスクリプト
├── server.py               # ローカルサーバー（自動更新対応）
├── config.json             # ★ あなたの設定（.gitignore済み）
├── config.json.example     # 設定テンプレート
├── ryunyu_foods.json       # 離乳食導入記録（.gitignore済み）
├── ryunyu_foods_example.json  # 導入記録テンプレート
├── data/                   # ぴよログエクスポートファイル（.gitignore済み）
│   └── sample.txt          # 動作確認用サンプルデータ
└── .gitignore
```

## 注意事項

- `data/` フォルダ内のエクスポートファイルと `ryunyu_foods.json`、`config.json` は `.gitignore` に含まれており、Gitの管理対象外です。個人情報をリポジトリにコミットしないようご注意ください。
- 本スクリプトはぴよログのエクスポート機能（公式提供）を利用しています。ぴよログのサーバーへのアクセスは行いません。
- 医療的な判断の根拠として使用しないでください。お子さんの健康状態は医師にご相談ください。
